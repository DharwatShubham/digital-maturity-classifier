\# Digital Maturity Classifier



\## 🚀 Overview



The \*\*Digital Maturity Classifier\*\* is a Machine Learning application that predicts an organization's \*\*Industry 4.0 Digital Maturity Stage\*\* based on organizational characteristics. The project uses a trained classification model to help assess digital transformation readiness.



\---



\## ✨ Features



\* Predicts Digital Maturity Stage

\* Machine Learning Classification

\* Dataset Generation

\* Model Training \& Evaluation

\* Fast Predictions using a Trained Model

\* Streamlit-based User Interface

\## ⚙️ Installation



Clone the repository:



```bash

git clone https://github.com/DharwatShubham/digital-maturity-classifier.git

```



Move into the project:



```bash

cd digital-maturity-classifier

```



Install dependencies:



```bash

pip install -r requirements.txt

```



Run the application:



```bash

streamlit run app.py

```
